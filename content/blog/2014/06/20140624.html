title=Corrigindo o erro: "method must have signature string method()" no Eclipse date=2014-06-24 type=post tags=blog status=published ~~~~~~

Corrigir erro apresentado pelo eclipse - "Method must have signature string method"

  • Primeiro: Acesse o eclipse e clique em Window - Preferences


  • Acesse: Web - JavaServer Faces Tools - Validation


  • Agora acesse: Type Assignment Problems - Method expression signature incompatibility
  • Marque essa última opção com: Warning ou Ignore


Feito isso, reinicie o eclipse e o erro estará corrigido.
