Instalar JRebel no Eclipse


Disponibilizada uma licença para DEV do JRebel.
  • Acessar o endereço: myJRebel
  • Depois de criar o cadastro, ir a aba Install and Activate, para liberar uma licença.


  • Feito isso, deve copiar a chave para jrebel no eclipse. Se não tiver instalado, ir em: Help-Eclipse MarketPlace e procurar por "JRebel"

  • No meu caso, eu já tenho instalado. Feito, então abrir a perspectiva do JRebel para ativar a chave.

  • Nessa perspectiva, é possível também, escolher o server que será monitorado pelo JRebel.

Feito isso, basta colar a chave disponibilizada e passar a usar o JRebel para o hot deploy.
